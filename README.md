## 🔹 What is FastAPI?

FastAPI is a **modern, fast (high-performance)** Python framework used to build APIs.
It is easy to learn and very powerful.

---

## 🔹 Why FastAPI?

* ⚡ Very fast performance
* 📄 Automatic API documentation
* 🧠 Easy to understand
* 🐍 Based on Python

* FastAPI basics

* Automatic Swagger UI
